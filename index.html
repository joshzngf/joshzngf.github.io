JPTT

JPTT is an app that makes it easier to browse PTT, the biggest forum in Taiwan.

The original PTT is an old telnet BBS, which requires users to use a terminal emulator to browse through. With JPTT, users can browse with ease using the intuitive touch interface. Also, JPTT provides functionalities that the original PTT does not offer, such as adding articles to bookmarks, or getting notified when someone responded to the article you followed. The Android version of the app is developed with Java, while the iOS version is developed with Swift. The back-end server enabling cloud features such as trending articles and bookmarks backup is built with Node.js.

JPTT currently has more than 450,000 downloads, 170,000 monthly active users, and 100,000 daily active users. It is currently on Google Play and App Store.

  • Links: Google Play / App Store / Website (Mandarin)
  • Since August 2013

Emergency Social Network

ESN is a mobile-centered social network website that allows people to communicate with one another in emergency situations. This end-to-end website is built with Node.js, PostgreSQL, and Angular.js. Unit tests, integration tests and continuous integration are used to achieve better robustness. Static analysis is applied with JSHint and ESLint to ensure code quality. Scrum and kanban are applied for project management during the development.

This app is selected as "The Best App" of the semester. This app also earned "Great UI," "Best Collaboration," "Best Functionality" badges.


Twitter Analytics on the Cloud

Designed and implemented Extract, Transform, Load on a 1TB Twitter raw dataset using MapReduce on AWS, GCP, Azure to support five types of queries. Efficient database schemas are designed and the data is then loaded into MySQL and HBase. Front-end servers are built and fine-tuned using Java. The whole system is deployed to AWS for live testing. To further improve perfomance, techniques such as load balancers, database sharding and replication are applied to improve the number of responses per second (RPS).

Ranked 6th out of 66 groups during the live test. Also one of the only 6 groups that fulfilled all the RPS baselines.
